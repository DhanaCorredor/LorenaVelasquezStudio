# Lorena Velásquez Studio

Sitio web del estudio de manicura premium y academia de Lorena Velásquez (Moratalaz, Madrid).

Stack: Vite + React 18 + TypeScript + Tailwind CSS + lucide-react.

## Desarrollo

```bash
npm install
npm run dev    # arranca el servidor en http://localhost:5173
npm run build  # genera /dist para producción
npm run preview
```
